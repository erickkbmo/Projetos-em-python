# 🔢 Verificador de Par ou Ímpar

Projeto desenvolvido em Python para praticar lógica de programação.

## 📋 Funcionalidades

- ✅ Verifica se um ou vários números são pares ou ímpares.
- ✅ Aceita vários números separados por espaço.
- ✅ Conta a quantidade de números pares.
- ✅ Conta a quantidade de números ímpares.
- ✅ Exibe o maior número informado.
- ✅ Exibe o menor número informado.

---

## 🚀 Como executar

1. Execute o arquivo Python.
2. Digite um ou mais números separados por espaço.

### Exemplo

**Entrada**

```text
10 3 8 15 20
```

**Saída**

```text
O número 10 é par.
O número 3 é ímpar.
O número 8 é par.
O número 15 é ímpar.
O número 20 é par.

Total de números pares: 3
Total de números ímpares: 2
Maior número: 20
Menor número: 3
```
